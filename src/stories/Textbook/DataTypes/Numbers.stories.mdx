[Ввод числового значения](https://learn.javascript.ru/task/repeat-until-number)

```ts
const readNumber = (): null | number => {
    const number = prompt('Введите число');
    if (!number) {
      return null;
    }
    if (Number.isFinite(+number)) {
      return +number;
    }
    return readNumber();
};
```

[Случайное число от min до max](https://learn.javascript.ru/task/random-min-max)

```ts
const random = (min: number, max: number) => min + Math.random() * (max - min);
```

[Случайное целое число от min до max](https://learn.javascript.ru/task/random-int-min-max)

```ts
const randomInteger = (
    min: number,
    max: number,
  ) => Math.floor(min + Math.random() * (max + 1 - min));
```
