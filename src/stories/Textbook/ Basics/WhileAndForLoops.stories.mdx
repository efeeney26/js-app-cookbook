[Выведите чётные числа](https://learn.javascript.ru/task/for-even)

```ts
for (let i = 2; i < 10; i++) {
    if (i % 2 === 0) {
      console.info(i);
    }
}
```

[Замените for на while](https://learn.javascript.ru/task/replace-for-while)

```ts
let k = 0;
  while (k < 3) {
    alert(`number ${k}!`);
    k++;
}
```

[Повторять цикл, пока ввод неверен](https://learn.javascript.ru/task/repeat-until-correct)

```ts
while (true) {
    const number = prompt('Введите число больше 100');
    if (!number || +number > 100) {
      break;
    }
}
```
